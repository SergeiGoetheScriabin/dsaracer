# dsa racer

simple typing trainer for practicing dsa code while improving wpm

## install

```bash
git clone https://github.com/sergeiGoetheScriabin/dsaracer.git
```

## run

open the html file in your browser and start typing

## planned features

* choose custom templates / coding languages
* leaderboard for highest wpm per game
* websocket multiplayer support for real time 1v1 typing races
* add more algorithms
